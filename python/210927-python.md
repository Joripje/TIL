## 오늘 배운 것

### 데이터 크롤링

- 동적페이지 크롤링
    - 개발자 도구에서 동적페이지의 숨겨진 url 찾음
    - url 접근 차단시 header를 추가하여 접근 요청(referer, user_agent)
    - request method를 확인하여 웹페이지 크롤링
    - json 데이터의 경우 `json.loads(response.text)` 를 사용하여 데이터 변환

- API
    - API를 활용해 수집된 데이터의 활용, 프로그램의 사용
    - 각 API의 레퍼런스, 예제 참고
